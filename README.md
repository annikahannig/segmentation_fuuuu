fuuuu.so
=======

## strsignal replacement for Segmentation Fault


### Usage

#### Text-Only-Version

$ LD_PRELOAD=./fuuuu.so bash

$ ./segfault_test
 
Segmentation F"fffffuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuu!

#### ASCII-Art-Version

$ LD_PRELOAD=./fuuuu-ascii.so bash

$ ./segfault_test
